HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son la base de cualquier página web. HTML proporciona la estructura del contenido, mientras que CSS se encarga de la presentación visual.

## HTML: La Estructura

Con HTML, defines los elementos de tu página: encabezados, párrafos, imágenes, enlaces, listas, etc. Es como el esqueleto de tu sitio.

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo.</p>
</body>
</html>
```

## CSS: El Estilo

CSS te permite dar vida a tu HTML, controlando colores, fuentes, tamaños, espaciados y la disposición de los elementos.

```css
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}
h1 {
    color: #0056b3;
}
```

¡Con estos dos lenguajes, las posibilidades son infinitas!
